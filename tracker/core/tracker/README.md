# Installing
To install the most recent stable version:

### yarn
```sh
yarn add @objectiv/tracker-core
```

### npm
```sh
npm install @objectiv/tracker-core
```

# Usage

```javascript
import { Tracker } from '@objectiv/tracker-core';
const tracker = new Tracker();
```
