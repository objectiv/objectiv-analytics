# Installing
To install the most recent stable version:

### yarn
```sh
yarn add @objectiv/tracker-web
```

### npm
```sh
npm install @objectiv/tracker-web
```

# Usage

```javascript
import { WebTracker } from '@objectiv/tracker-web';
const webTracker = new WebTracker({ endpoint: '/collector' });
```
